# EaseMob SDK Examples Android ##



## ChatDemoNonUI ##
本demo展示了怎样使用易聊SDK快速创建一个类微信聊天APP。展示的功能包括：易聊SDK初始化，登录，登出，注册消息接收listener, 发送消息

## ChatDemo ##
本demo展示了怎样使用易聊SDK快速创建一个完整的类微信聊天APP。展示的功能包括：注册新用户，用户登录，添加好友，单聊，群聊，发送文字，表情。
